# Plaid Sync Addon

A Wealthfolio addon for syncing financial accounts with Plaid.

## Features

- Connect financial institutions via Plaid
- Automatically sync transactions and balances
- Secure account management

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev:server

# Build for production
pnpm build

# Package for distribution
pnpm bundle
```

## Usage

1. Install the addon in Wealthfolio
2. Navigate to "Plaid Sync" in the sidebar
3. Follow the instructions to connect your accounts
